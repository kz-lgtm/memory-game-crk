# memory-game-crk

## Overview
An interactive memory matching game themed around Cookie Run characters, where players flip cards to find matching pairs.

## Tech Stack
- HTML
- CSS
- JavaScript

## Features
- Card flipping animations
- Matching logic for pairs
- Restart button to replay the game
- Custom themed visuals using character images

## What I Learned
- Implementing game logic using JavaScript
- Handling user interactions with event listeners
- Dynamically updating the UI based on game state
- Working with images and custom assets

## Notes
This project was inspired by a tutorial and customized with a new theme, styling, and adjustments to better understand the underlying logic.
